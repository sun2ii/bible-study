# Matthew - King

## At a Glance

| | |
|---|---|
| **Author** | Matthew (Levi), tax collector turned disciple |
| **Audience** | Jewish believers |
| **Theme** | Jesus is the promised Messiah-King |
| **Key Phrase** | "This was to fulfill what was spoken..." |
| **OT Quotes** | ~60 references |
| **Unique Term** | "Kingdom of Heaven" (32x) |

## The Big Picture

```
MATTHEW'S QUESTION
    │
    └─► "Is this the King Israel has waited for?"
        │
        ├─ Ch 1-4:   The King ARRIVES    → Credentials proven
        ├─ Ch 5-7:   The King TEACHES    → Kingdom ethics
        ├─ Ch 8-10:  The King ACTS       → Authority demonstrated
        ├─ Ch 11-13: The King DIVIDES    → Accepted or rejected
        ├─ Ch 14-20: The King REVEALS    → True identity
        ├─ Ch 21-25: The King CONFRONTS  → Final conflict
        └─ Ch 26-28: The King CONQUERS   → Death & resurrection
```
